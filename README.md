# Personal-Budget-Tracker
## Description
A Java console program for tracking personal expenses for my PP1 Honors Project.

## How to Run
1. Compile all `.java` files inside the `src` folder.
2. Run the `Main` class.
3. Use the menu to add expenses, view expenses, and calculate totals.

---

## Features
- Add new expenses with amount, category, and description
- View all recorded expenses
- Calculate total spending
- Organized using OOP (Expense + BudgetManager)
- ArrayList used for dynamic data storage
- Menu-driven interface using loops and selections
- Expandable for future features (file I/O, category summaries, etc.)

---

## Concepts Used (PP1 Alignment)
- Variables, expressions, input handling
